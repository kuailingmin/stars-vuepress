# 介绍

Stars Use 是 Stars 团队提供的 Vue3.x Composition API 库。

## 特性

- 从实际业务应用场景中沉淀
- 基于 TypeScript 开发
- 完善的单元测试，提供稳定性保障

## 安装

```bash
# 通过 npm 安装
npm i stars-use -S

# 通过 yarn 安装
yarn add stars-use
```
