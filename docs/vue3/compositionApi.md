# vue composition API

::: tip 什么是组合API
 在vue3中引入的一种新的编写Vue组件的方式
:::

**Vue2  VS  Vue3**

| 2.x(对象式API)                | 3.x(组件式API)                           |
| -------------                |:---------------:                        |
| 不利于复用                     | 极易复用(原生js)                          |
| 潜在的命名冲突                  | 可灵活组合 （生命周期钩子可多次使用）         |
| 上下文丢失                     | 提供更好的上下文支持                       |
| 有限的类型支持                  | 更好的TypeScript支持                     |
| 按API类型组织                  | 按功能或逻辑组织（提供大型项目维护成本）       |
|                              | 可独立于Vue组件使用       |

**什么是可组合的函数**
::: tip 说明
 可复用逻辑的集合，专注点分离,不同于react hooks,Vue3的‘setup()’只会在组件建立时执行一次，并建立数据与逻辑之间的连结。
:::
