# 介绍

Stars Ui 是 Stars 团队提供的 Vue3.x UI组件库。

## 特性

- 给开发者几乎所有的机会来自定义组件，满足一切项目使用场景
- 配合vue3的composition API，更快的相应
- 组件体积更小，灵活使用，按需使用
- 完善的单元测试，提供稳定性保障

## 安装

```bash
# 通过 npm 安装
npm install  @kuailingmin/stars

# 通过 yarn 安装
yarn add @kuailingmin/stars
```


资料参考：[Stars ui](http://starsui.com/stars-ui/index.html#/)
